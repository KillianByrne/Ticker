Dashboards Sample Build Process

=====================================================
Prerequisites
=====================================================
node.js must be installed (see https://nodejs.org/)

In the project _build directory, run:
npm install

This will install the necessary packages that support
the build.

=====================================================
Building Sample Project
=====================================================
In the project _build directory, run:
gulp

This will compile the files from the source folder
into the apps folder, wrapping them in a Require
module
